Descripción de la tarea número: 43
==================================================
Reto: (Requerimientos)
1.- Diseñar una estructura básica de HTML y verla en el navegador.
2.- Introducir algo de color a la página apoyándose en CSS.
3.- Añádir un video de youtube gracias a la nueva etiqueta video.
4.- Serías capaz de darle al play y al pause del video mediante instrucciones por voz (hablando a la computadora) utilizando javascript? (no obligatorio).
Solución propuesta: Hemos construido un sencillo HTML en el que hemos insertado un video, dos botones y una explicación de cómo funciona el ejemplo. El estilo de toda la página se lo hemos dado adjuntando una hoja destilo de código abierto. El código javascript lo hemos tomado de un desarrollo similar de Ian Devlin (ver: https://iandevlin.com/examples/speechapi-video/). La funcionalidad requerida en el punto 2 la hemos hecho con un sencilo evento que cambia las etiquetas CSS de los dos botones de acuerdo a sendos event listeners de javascrit.
Instrucciones de uso: 
1- Descargar los 4 archivos a una misma carpeta en tu equipo local, a saber:
- La página principal "index.html"
- El video "Clouds.mp4"
- El archivo que contiene todo el codigo javascript "speech-recog-2.js"
- La hoja de estilos "style.css"
2- Abrir, desde el directorio local, el archivo index.html con el navegador Chrome
3- Una vez abierta la página en el navagedor, seguir las instrucciones que se dan en ella para probrar el funcionamiento del reconocimiento de voz en HTML5.

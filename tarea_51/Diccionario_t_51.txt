Descripción: CPU
Descripción: CPU son las siglas de Central Processing Unit, Unidad Central de Procesamiento. Comúnmente se le suele denominar microprocesador. Es la pieza fundamental de la computación pero no se encarga a pesar de ello de realizar todas las tareas (muchas quedan delegadas en otros componentes como la RAM, la tarjeta gráfica, etc); así su labor tiene más que ver con la coordinación y dirección de los cálculos realizados en el sistema.

Descripción: Periférico
Descripción: Son los componentes auxiliares a las tareas de procesamiento y cálculo de procesador, mediante ellos se provee de datos de entrada y salida al sistema, capacidad de almacenamiento de los datos, así como otras funciones básicas de comunicación necesarias para que la computadora se pueda comunicar con otras computadoras.

Tarea: 51
Id: 41
Descripción: Terminal
Descripción: En informática, se denomina terminal o consola a un dispositivo electrónico, electromecánico o computacional (revisar) que se utiliza para interactuar con un computador. El ejemplo más cercano de ello sería un terminal ligero que, con menos capacidad de proceso y con una configuración básica (incluso de sistema operativo), es capaz de ejercer de medio de entrada a un servidor. Terminal en el entorno Linux se refiere al interfaz textual para manejar el sistema mediante líneas de comando.

Tarea: 51
Id: 42
Descripción: Interfaz
Descripción: Interfaz (interfaces, en plural) se refiere en informática a un subsistema que permite la conexión funcional entre dos sistemas, programas, dispositivos o componentes de cualquier tipo que proporciona una comunicación de distintos niveles, permitiendo el intercambio de información. Cuando la conexión es un computador y un usuario nos referiremos a interface de usuario.
